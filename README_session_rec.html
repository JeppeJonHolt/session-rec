session-aware

Requirements

To run session-rec, the following libraries are required:
  • Anaconda 4.X (Python 3.5+)
  • BLAS
  • Certifi
  • CUDA
  • Dill
  • NetworkX
  • NumExpr
  • NumPy
  • Pandas
  • Psutil
  • Pympler
  • PyTables
  • Python-dateutil
  • Python-telegram-bot
  • Pytz
  • Pyyaml
  • Scikit-learn
  • SciPy
  • Six
  • Sklearn
  • Tensorflow
  • Tensorflow-gpu
  • Theano

Installation

Using docker (Linux users with a GPU that supports Cuda9)

  1. docker image: wolfgangschmid/tensorflow2_theano_python3_miniconda3:0.1
  2. conda install --yes --file requirements_conda.txt
  3. conda install --yes --file requirements_hgru.txt
  4. pip install -r requirements_pip.txt
